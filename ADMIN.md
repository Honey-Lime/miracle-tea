# Админка для Miracle Tea

## Настройка прав администратора

Права администратора теперь выдаются по email из переменной `ADMIN_EMAILS` в `backend/.env`.

Пример:

```env
ADMIN_EMAILS=admin1@yandex.ru,admin2@yandex.ru
```

Если нужно вручную выдать права первому email из списка, выполните команду:

```bash
cd backend
node setAdmin.js
```

## Вход в админку

1. Войдите в аккаунт с email, который указан в `ADMIN_EMAILS`
2. В хидере справа появится круглая оранжевая кнопка "Adm"
3. Нажмите на неё для входа в панель администратора

## Страницы админки

### Заказы

- Отображает все оформленные заказы
- Каждый заказ содержит:
  - Дату и время оформления
  - Имя и email клиента
  - Список товаров с ценами
  - Стоимость чая
  - Стоимость доставки
  - Адрес доставки
  - Итоговую сумму

### Редактирование

#### Вкладка "Добавить чай"

Форма для добавления нового товара:

- Название
- Содержимое (через запятую)
- Описание
- Цена (₽)
- Себестоимость (₽)
- Остаток на складе (г)

После нажатия кнопки "Добавить" товар сохраняется в базе данных.

#### Вкладка "Поставка"

1. Введите название чая в поле поиска
2. Нажмите "Найти"
3. Выберите чай из списка результатов
4. Введите количество добавляемых грамм
5. Нажмите "Добавить"

Количество чая на складе будет увеличено.

## Уведомления

Все операции в админке сопровождаются уведомлениями в нижней части экрана:

- ✅ Зеленое - успешное выполнение
- ❌ Красное - ошибка
- ⚠️ Оранжевое - предупреждение

## Структура файлов

### Backend

- `backend/src/models/User.js` - добавлено поле `isAdmin`
- `backend/src/middleware/adminMiddleware.js` - проверка прав администратора
- `backend/src/controllers/adminController.js` - контроллер для админ-операций
- `backend/src/routes/adminRoutes.js` - роуты для админки
- `backend/setAdmin.js` - скрипт для установки прав администратора

### Frontend

- `frontend/src/pages/AdminPanel.jsx` - главная страница админки
- `frontend/src/pages/admin/OrdersPage.jsx` - страница заказов
- `frontend/src/pages/admin/EditPage.jsx` - страница редактирования
- `frontend/src/pages/admin/AddTeaTab.jsx` - вкладка добавления чая
- `frontend/src/pages/admin/SupplyTab.jsx` - вкладка поставки
- `frontend/src/components/Header.jsx` - обновлен для отображения кнопки админки
