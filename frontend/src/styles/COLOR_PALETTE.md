# Цветовая палитра Miracle Tea

## Основная концепция

Luxury-сегмент с благородными глубокими цветами. Золотые акценты в паре с темно-синим создают классическое сочетание, подчеркивающее статус магазина и ценность чая.

## Цвета

### Основной фон (Primary)

- **Чистый белый**: `#FFFFFF`
- **Легкий подтон**: `#F8F9FA`

Используется для основного фона сайта и карточек товаров. Светлый фон обеспечивает отличную читаемость.

### Основной акцент (Brand Color)

- **Полуночный синий**: `#1A3B5D`
- **Темный вариант**: `#0F2438`
- **Светлый вариант**: `#2A4D6F`

Благородный глубокий цвет индиго. Используется для:

- Шапки сайта
- Футера
- Акцентных кнопок
- Активных элементов навигации

### Второстепенный акцент (Secondary)

- **Steel Blue**: `#4A7A9C`
- **Темный вариант**: `#3A5F7A`
- **Светлый фон**: `#E8F1F5`

Используется для:

- Наведения мышки (hover)
- Иконок категорий
- Второстепенных акцентов

### Текст (Text)

- **Основной текст**: `#2C3E50` (темно-сизый)
- **Второстепенный текст**: `#7F8C8D` (серый для описаний)

### Акцент для динамики (CTA)

- **Золото**: `#D4AF37`
- **Темное золото**: `#B8952B`
- **Светлое золото**: `#F4E5B0`

Металлик/золото для CTA кнопок и акцентов. В паре с темно-синим создает luxury-эффект. Используется для:

- Кнопок призыва к действию
- Цен товаров
- Важных акцентов

## CSS переменные

```css
/* Primary Colors - Background */
--color-primary: #ffffff;
--color-primary-subtle: #f8f9fa;

/* Brand Color - Main Accent */
--color-brand: #1a3b5d;
--color-brand-dark: #0f2438;
--color-brand-light: #2a4d6f;

/* Secondary Color */
--color-secondary: #4a7a9c;
--color-secondary-dark: #3a5f7a;
--color-secondary-light: #e8f1f5;

/* Text Colors */
--color-text-primary: #2c3e50;
--color-text-secondary: #7f8c8d;

/* CTA Accent - Dynamic */
--color-cta: #d4af37;
--color-cta-dark: #b8952b;
--color-cta-light: #f4e5b0;
```

## Примеры использования

### Кнопки

- **Основные кнопки**: `--color-brand` (фон), `--color-white` (текст)
- **CTA кнопки**: `--color-cta` (фон/акцент), `--color-brand` (текст)
- **Hover эффект**: `--color-brand-dark` или `--color-cta-dark`

### Карточки товаров

- **Фон**: `--color-primary` (белый)
- **Граница**: `--border-color` (#E0E0E0)
- **Тень**: `--shadow-color` (rgba(26, 59, 93, 0.1))

### Цены

- **Обычная цена**: `--color-text-primary`
- **Акцентная цена/Скидка**: `--color-cta`

### Статусы заказов

- **Оплачено**: `--color-cta` / `--color-cta-light`
- **В доставке**: `--color-secondary` / `--color-secondary-light`
- **Завершен**: `--color-secondary` / `--color-secondary-light`
- **Отменен**: `#DC5656` / `#FDE8E8`
