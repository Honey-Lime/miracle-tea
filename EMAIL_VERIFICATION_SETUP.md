# Настройка email-верификации

## 1. Что нужно настроить

Откройте `backend/.env` и заполните SMTP-параметры.

Пример для Yandex Mail:

```env
SMTP_HOST=smtp.yandex.ru
SMTP_PORT=465
SMTP_USER=your-login@yandex.ru
SMTP_PASS=your-app-password
EMAIL_FROM_ADDRESS=your-login@yandex.ru
EMAIL_FROM_NAME=Miracle Tea
```

## 2. Как это работает

### Регистрация

1. Пользователь вводит email
2. Нажимает "Получить код"
3. Получает письмо с 6-значным кодом
4. Вводит код
5. Завершает регистрацию

### Восстановление пароля

1. Пользователь вводит email
2. Получает письмо с кодом
3. Вводит код и новый пароль

## 3. Защита от злоупотреблений

- Код действителен 5 минут
- Повторная отправка кода возможна через 1 минуту
- Использованные коды помечаются и не могут быть использованы повторно
- Просроченные коды автоматически удаляются

## 4. Логи

При отправке письма backend пишет в лог:

- начало отправки;
- успешную отправку;
- текст ошибки SMTP, если отправка не удалась.

## 5. Важное

Для реальной отправки должен быть установлен пакет:

```bash
npm install nodemailer
```
