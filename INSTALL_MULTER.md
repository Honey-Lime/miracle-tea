# Установка зависимостей для загрузки файлов

## Необходимые действия

Для работы нового функционала загрузки изображений и видео необходимо установить пакет `multer`:

### Вариант 1: Через командную строку

1. Откройте командную строку (cmd) или PowerShell **от имени администратора**
2. Выполните команду:

```bash
cd c:\Users\vikdub\Documents\GitHub\miracle-tea\backend
npm install multer
```

### Вариант 2: Если PowerShell блокирует выполнение

Если вы видите ошибку о политике выполнения скриптов, выполните:

```bash
cd c:\Users\vikdub\Documents\GitHub\miracle-tea\backend
node "C:\Program Files\nodejs\node_modules\npm\bin\npm-cli.js" install multer
```

Или просто скопируйте файл `node_modules` из другого проекта где установлен multer.

### Вариант 3: Добавить в package.json и запустить run.bat

1. Откройте `backend/package.json`
2. Добавьте в зависимости `"multer": "^1.4.5-lts.1"`
3. Запустите `run.bat` - npm автоматически установит зависимости

## Проверка установки

После установки проверьте что файл существует:

```
backend/node_modules/multer/package.json
```

## После установки

Запустите проект через `run.bat`. Файлы будут сохраняться в:

```
backend/uploads/products/
```

Эта директория создается автоматически при первом запуске.

## Примечание

Если у вас уже запущен проект, остановите его и перезапустите после установки multer.
